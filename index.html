Twitter Sentiment Analysis

This was a team project in which we created a site to analysis sentiment on topics being discussed on Twitter. It displays trending topics for the city, state entered and you can also search for any topic you wish. It then runs all the tweets returned through a sentiment analysis API to get the overall sentiment about the topic. Utilizies Node.js for Twitter authentication and request handling as well as the Twitter API.
